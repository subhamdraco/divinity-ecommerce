# Divinity Impex — Global Brands Business Hub

Premium corporate hub website for Divinity Impex, showcasing four business segments with dedicated routes.

## Routes

| Route | Segment |
|-------|---------|
| `/` | Hub homepage |
| `/titan-core` | Sports Supplement Range |
| `/reshu` | FMCG / Personal Care |
| `/nova` | NOVA Inner Compass |
| `/nova/corporate` | Corporate leadership programs |
| `/nova/schools` | Schools & student development |
| `/rizwan-adatia` | Rizwan Adatia portfolio |

## Development

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16 (App Router)
- Tailwind CSS 4
- Framer Motion
- Lucide React icons

## Design

- Hub: Navy `#0D1B2A` + Gold `#C9A227`
- Typography: Playfair Display, Inter, Montserrat
- Mobile-first responsive layout
